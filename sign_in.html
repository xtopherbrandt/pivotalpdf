Agile Docs creates printable output from Pivotal Tracker stories.

Agile Docs creates well formatted PDF documents from a selection of Pivotal Tracker user stories. Sign in with your Pivotal Tracker API Token. Once authenticated select your project and the stories you want to include then hit Generate PDF.

The Agile Docs web application includes multiple format options. The Full Document format uses the entire description of each story and also include story activity notes. The Summary Document format only includes the first paragraph of each story.

For even greater convenience, check out the Agile Docs Chrome Extension.. The extension adds a PDF button to the multiple stories selection tools (CSV, clone, delete...) in Pivotal Tracker. It doesn't get any easier!
